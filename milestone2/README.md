## 快速开始
### 本地开发运行

```bash
# 克隆仓库
git clone https://github.com/jane872/module3.git
cd module3
# 安装依赖
pip install -r app/requirements.txt
# 启动服务
python app/app.py
# 或使用docker-compose本地启动
docker-compose up -d
